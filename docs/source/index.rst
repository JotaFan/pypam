Pypam |version|
===============


Description
-----------

The goal of pypam is to allow easy reading and processing of acoustic
underwater data. Pypam further depends on `pyhydrophone`_ for hydrophone 
data management.

.. _pyhydrophone: https://github.com/lifewatch/pyhydrophone
